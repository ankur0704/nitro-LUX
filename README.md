## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` 
3. first create a file `.env.local`
4. paste your Gemini API key ->  GEMINI_API_KEY=your_actual_api_key_here
5. Run the app:
   `npm run dev`
